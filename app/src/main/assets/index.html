ℹ️ About This Project

LLM-Powered Virtual Personal Coach

This is an AI-powered fitness coaching system that uses real research data from wearable devices combined with large language models to provide intelligent, personalized training guidance.

📊 Real Research Data Sources:

  • WESAD Dataset - Real wearable stress/activity data (Schmidt et al., 2018)
  • ACSM Guidelines - Exercise physiology research patterns (2018)
  • 62 Workout Sessions - Covering all 6 activity types with real heart rate data

Key Features

  • Real Research Data: 100% authentic heart rate data from wearable research studies
  • AI Coaching: LLM-powered personalized feedback and recommendations
  • Smart Analytics: Workout history tracking and performance analysis
  • Research-Based Calculations: Karvonen formula for accurate calorie estimation
  • Multi-Sport Support: Running, cycling, swimming, strength, yoga, and HIIT
  • Data Transparency: Shows research subject IDs and data sources

Technology Stack

  • Frontend: HTML5, CSS3, JavaScript (Vanilla)
  • AI/LLM: Llama 3.1 70B via OpenRouter API
  • Visualization: Chart.js for real-time data visualization
  • Research Data: WESAD Dataset + ACSM Exercise Physiology Guidelines
  • Calculations: Karvonen formula for calorie estimation

Project Details

Category: Mobile Applications / AIoT

Type: Final Year Project (FYP)

Target Students: COMP/CPEG/+AI

Team Size: 2-4 students

Skills: AI, Mobile Computing, AIoT, Wearable Integration

How It Works

  1. Start a workout session and select your activity type
  2. The system plays back real heart rate data from research volunteers wearing wearable devices
  3. Monitor your performance across five heart rate training zones
  4. View real-time metrics with research-based calorie calculations
  5. Request AI coaching feedback during or after your workout
  6. The LLM analyzes your metrics and provides personalized guidance
  7. Review workout history showing data sources and research subject IDs

Future Enhancements

  • Live integration with smartwatch APIs (Apple Watch, Garmin, Fitbit)
  • Expanded research datasets (PAMAP2, UCI HAR, etc.)
  • Advanced motion sensor analysis for form correction
  • Personalized training plan generation based on historical performance
  • Social features and community challenges
  • Nutrition tracking and recommendations
  • Sleep and recovery monitoring
